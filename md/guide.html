快速上手

本文将会帮助您从头搭建此项目

环境准备

  • 本项目推荐使用 VSCode 进行开发。
  • 本地环境需要安装Node.js 16.0+、pnpm7.0+、Git工具等。

代码拉取

从 Gitee 上拉取

 git clone https://gitee.com/sina_xys/m-admin.git

从 Github 上拉取

 git clone https://github.com/sina-xys-felix/M-Admin.git

安装使用

安装依赖

 // 使用pnpm安装依赖
 pnpm install

运行项目

 // 运行项目--开发环境
 pnpm dev
 // 运行项目--生成环境
 pnpm prod

打包项目

 // 打包项目
 pnpm build:prod

工具推荐

  • Nvm --> node版本管理工具
  • Volar --> vue3官方插件
  • Vue3 Snippets --> vue3代码提示工具
